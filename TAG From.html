The datalist Element

The datalist element specifies a list of pre-defined options for an input element.

Note: The datalist tag is not supported in Safari or IE9 (and earlier).

Tag yang terdapat pada example diatas


1. form Membuat formulir untuk mengumpulkan input pengguna
2. input Membuat tipe inputan pada form yang akan dibuat
3. option Mendefinisikan opsi yang bisa dipilih
4. datalist Membuat daftar pilihan untuk input data
